Question-1: What are the different ways to select an element in the DOM?
Different ways to select in the dom is For single elements: Use getElementById() or querySelector() and multiple elements: Use getElementsByClassName(), getElementsByTagName(), or querySelectorAll().
Question-2: What is the difference between innerHTML, innerText, and textContent ?
difference between innerHTML, innerText, and textContent is innerHTML Use when you need to manipulate HTML content inside an element And innerText Use when you need the visible text, considering styles and layout And innerContent Use for retrieving or modifying raw text, with no layout or styling considerations.
Question-3: What is event delegation in the DOM?
Event delegation allows you to handle events on a parent element rather than on individual child elements.It works by utilizing event bubbling, which lets the event bubble up from the target element to the parent element.
